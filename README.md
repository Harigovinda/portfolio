## How to Deploy This Portfolio

### Option 1: GitHub Pages (Free Hosting)

1. **Create a GitHub Repository:**
   - Go to [GitHub](https://github.com) and sign in
   - Your repository should already be created as: **dev-protfolio**
   - Make sure it's set to Public

2. **Upload Files:**
   - Clone the repository to your local machine:
     ```
     git clone https://github.com/YOUR_USERNAME/dev-protfolio.git
     ```
   - Copy all files from this portfolio folder to the repository
   - Commit and push:
     ```
     git add .
     git commit -m "Initial portfolio upload"
     git push origin main
     ```

3. **Enable GitHub Pages:**
   - Go to your repository on GitHub
   - Click "Settings"
   - Click "Pages" in the left sidebar
   - Under "Build and deployment" > "Branch":
     - Select "main" (or "master")
     - Select "/ (root)"
   - Click "Save"
   - Wait 1-2 minutes for deployment

4. **Get Your Live URL:**
   - Your portfolio will be live at: `https://YOUR_USERNAME.github.io/dev-protfolio/`

### Option 2: Deploy via Git Commands

If you already have Git installed:

```bash
# Initialize git if not already done
git init

# Add all files
git add .

# Commit with message
git commit -m "My Java Developer Portfolio"

# Add remote repository
git remote add origin https://github.com/YOUR_USERNAME/dev-protfolio.git

# Push to GitHub
git push -u origin main
```
